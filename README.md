## 📘 Descripción general
Consolida la adquisición de los siguientes aprendizajes:

IL7. Diseña jerarquías de clases que representen adecuadamente relaciones de herencia y composición entre objetos, de acuerdo con los requerimientos del sistema, asegurando la reutilización y extensibilidad del código. 

---

## 🧱 Estructura general del proyecto

```plaintext
📁 src/
├── ui/         # Clase principal con el método main
├── model/      # Clases de dominio (ExcursionCultural-SubClase, PaseoLacustre-SubClase, RutaGastronomica-SubClase, ServicioTuristico-SuperClase)
└── data/       # Clase gestor de datos
````
